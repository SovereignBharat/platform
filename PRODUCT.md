# SovereignBharat Platform

India's sovereign cloud and AI-native infrastructure platform.

## Product Thesis

SovereignBharat is designed as a sovereign cloud operating layer for India: compute, storage, networking, Kubernetes, GPU infrastructure, AI inference, agent runtimes, governance, and developer tooling in one platform.

## Core Platform Areas

- Sovereign cloud infrastructure
- GPU and AI compute fabric
- Kubernetes-native orchestration
- AI inference infrastructure
- Agent runtime and workflows
- Governance and policy systems
- Developer SDKs and APIs
- Enterprise and government deployments

## Suite

### SovereignBharat Cloud
Compute, networking, storage, and Kubernetes.

### SovereignBharat AI
Inference APIs, embeddings, model serving, and vector infrastructure.

### SovereignBharat Runtime
Agents, orchestration, workflows, scheduling, and event execution.

### SovereignBharat Guard
Security, governance, policy, IAM, and audit.

### SovereignBharat Console
Infrastructure and AI management dashboard.
