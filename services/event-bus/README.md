# SovereignBharat Event Bus

Event-driven backbone for the platform.

## Responsibilities

- Runtime events
- Deployment events
- Workflow coordination
- Observability streaming
- Audit propagation
- Multi-service orchestration

## Suggested Technologies

- NATS
- Kafka
- Redpanda

## Event Examples

```txt
cluster.created
workflow.started
workflow.failed
agent.executed
model.deployed
gpu.allocated
policy.denied
```
