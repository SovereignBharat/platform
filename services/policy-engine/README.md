# SovereignBharat Policy Engine

Governance and policy enforcement layer for sovereign cloud and AI runtime operations.

## Responsibilities

- Runtime policy evaluation
- RBAC and permission checks
- Deployment guardrails
- Agent execution boundaries
- Audit event generation
- Approval workflow enforcement
- Data residency controls

## Core Policy Domains

- Identity
- Infrastructure
- Deployments
- Models
- Agents
- Workflows
- Data
- Secrets

## Decision Model

```txt
allow | deny | require_approval | require_review
```
